Bootstrap Features

This website uses multiple aspects of the bootstrap library in conjunction with the AngularJS UI-Bootstrap API. Overview topics include Grid system, Grid hide show implementations, combination with bootstrap social for social-icons, UIB-modal, floating, panel containers, blockquotes, text and typography.

Other API's / Libraries / Frameworks Used

JQuery Vibrate: Vibration for mobile devices when using the navigation bar. AngularJS: The webpage is build using a component hierarchy and makes http calls to retrieve work items allows for simple updates in the future. AngularJS is used to provide cached versions of all my pages to allow a quick load through navigation. A searh of work items is also possible using Angulars filter. NGTweet: An AngularJS plugin, used to embed my most recent tweet on the contact page. Formspring: Allows simple email sending from my contact form without the use of server side code on my part. Just HTML and Javascript. CSSTransitions: These are used heavily throughout the website to create a smooth user experience but not over used in order to maintain performance.
